# Security Policy

## Supported Versions

I will always support the latest commit and the previous minor version

| Version       | Supported          |
| ------------- | ------------------ |
| latest commit | :white_check_mark: |
| 1.2.3         | :white_check_mark: |
| 1.2.2         | :white_check_mark: |
| 1.2.1         | :white_check_mark: |
| 1.2.0         | :white_check_mark: |
| 1.1.0         | :x:                |
| 1.0.0         | :x:                |
